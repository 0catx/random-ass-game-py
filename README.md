# DOUBLE GUESS

A really dumb number guessing game :p

The way it works is the script generates two random numbers between 1 and 10 and the goal of the player is to guess those two numbers within a tolerance of 4.

This project was originally made for a statistics project of mine, so it comes with a SQLite database feature for keeping telly and why it seems kinda like a gambling game; wording wise.

## Usage

All you need is python.

To run the script, simply run:

`$ python game.py `

Note: It should make the DB file but if not, it'll run regardless.

###### I implemented checks to sure it can run, even if it can't telly and warn of such.
